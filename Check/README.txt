How to use AD Checker v1.0

1. Make sure the excel file is in the same directory as the AD Checker

2. Make sure the User Group Attribute is the last attribute to be compared. There can be extra attribute that is not compared at the end.

3. Change line separator to pipe (|) in Control Panel > Region and Language > Additional Settings, and export the excel file as .csv

4. Make sure the AD files are in the same directory, or as a child directory under the excel file's directory

5. Run the AD Checker and obtain result in "Check Result.txt"

Source Code of AD Checker can be pulled from github:
https://github.com/jimmylkm/AD-Checker.git
